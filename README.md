# 操作系统课程设计模拟系统

## 一、项目简介

本项目为《操作系统》课程设计，采用 C 语言实现操作系统中的典型功能模块，包括处理机调度、内存管理、进程同步与文件系统模拟等内容。

项目通过模拟实现经典操作系统算法，加深对操作系统核心原理、资源管理机制以及并发控制思想的理解。

开发环境为 Visual Studio 2022。

---

## 二、项目功能

### （1）处理机调度

实现了以下典型调度算法：

- 先来先服务（FCFS）
- 短作业优先（SJF）
- 时间片轮转（RR）
- 优先级调度（Priority）
- 高响应比优先（HRRN，拓展部分）

程序能够输出：

- 进程运行顺序
- 完成时间
- 周转时间
- 带权周转时间
- 平均周转时间
- 平均带权周转时间

便于比较不同调度算法的性能差异。

---

### （2）内存管理

实现了以下内存管理算法：

#### 动态分区管理

- 首次适应算法（First Fit）

实现内存申请与分配过程模拟。

#### 页面置换算法

- FIFO 页面置换算法
- LRU 页面置换算法

程序能够统计：

- 缺页次数
- 缺页率

用于分析不同页面置换策略的性能。

---

### （3）进程同步与并发控制

采用多线程方式模拟经典生产者—消费者问题。

实现内容包括：

- 生产者线程
- 消费者线程
- 共享缓冲区
- 互斥访问控制
- 信号量同步机制

能够正确实现线程间同步与互斥，避免数据竞争问题。

---

### （4）文件系统模拟

实现简单文件系统管理功能：

- 创建文件
- 写入文件
- 读取文件
- 删除文件
- 查看文件目录

通过模拟文件控制块（FCB）实现文件管理功能。

---

## 三、拓展部分

### HRRN（高响应比优先）调度算法

在基础调度算法的基础上，实现了高响应比优先调度算法（Highest Response Ratio Next）。

其响应比计算公式为：

R = (等待时间 + 服务时间) / 服务时间

该算法综合考虑等待时间和运行时间，在保证公平性的同时提高系统整体调度性能。

项目对 FCFS、SJF、RR、Priority 以及 HRRN 等算法进行了结果比较与分析。

## 八、项目总结

本项目通过模拟实现处理机调度、内存管理、进程同步和文件系统等操作系统核心模块，进一步理解了操作系统资源管理、调度策略以及并发控制等重要原理。
